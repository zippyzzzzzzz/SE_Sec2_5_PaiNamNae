# คู่มือผู้ใช้: รายงานผู้ขับขี่หลังจบทริป (Passenger → Driver)

## ภาพรวม
ผู้โดยสารสามารถส่งรายงานพฤติกรรม/เหตุการณ์ของผู้ขับขี่ได้ภายใน 3 วันหลังทริปจบ โดยต้องเป็นการเดินทางที่ผู้โดยสารเป็นผู้จองและสถานะเส้นทางเป็น COMPLETED

## เงื่อนไขก่อนใช้งาน
- ต้องเข้าสู่ระบบในบทบาทผู้โดยสาร
- ทริปต้องถูกยืนยันและกด "จบทริป" แล้วทั้งฝั่งผู้ขับและผู้โดยสาร
- ส่งรายงานได้ภายใน 3 วันหลังทริปจบเท่านั้น
- ต้องกรอกข้อมูลติดต่อ (ชื่อ, นามสกุล, เบอร์โทร, อีเมล)
- ต้องแนบสื่ออย่างน้อย 1 ไฟล์ (รวมสูงสุด 3 ไฟล์)
  - รูป: png/jpg/jpeg ขนาด ≤10MB/ไฟล์
  - วิดีโอ: mp4 ขนาด ≤30MB/ไฟล์

## ขั้นตอนการส่งรายงาน (ผู้โดยสาร)
1) ไปที่เมนูการแจ้งเหตุ/รายงาน
2) ระบบตรวจสอบสิทธิ์อัตโนมัติ (ต้องเป็นเจ้าของการจองและภายใน 3 วัน)
3) เลือกหมวดหมู่รายงาน เช่น INAPPROPRIATE_BEHAVIOR / LOST_ITEM / CLEANLINESS / OTHER / VIOLATED_AGREEMENT / DETAILS_MISMATCH
4) กรอกหัวข้อและรายละเอียดเหตุการณ์
5) กรอกข้อมูลติดต่อ (ชื่อ, นามสกุล, เบอร์โทร, อีเมล)
6) แนบไฟล์รูป/วิดีโอ (อย่างน้อย 1 ไฟล์ รวมไม่เกิน 3 ไฟล์)
7) กดส่ง ระบบจะบันทึกสถานะเริ่มต้นเป็น PENDING และส่งไปยังผู้ดูแลระบบ

## การติดตามผลรายงาน
- สถานะที่อาจพบ: PENDING → UNDER_REVIEW → CONTACTING_DRIVER → RESOLVED / CLOSED
- ผู้โดยสารดูรายงานของตนได้ที่หน้ารายงานของฉัน (My Reports)
- สามารถกรองตามสถานะหรือหมวดหมู่ได้
