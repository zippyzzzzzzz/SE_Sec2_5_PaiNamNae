# UAT Verification - Test Data

Input data for verification tests includes the following kinds of elements:

- **Documents**: photos of national ID cards or driver licenses. Variants include clear, blurred, skewed, or deliberately invalid cards.
- **Selfies**: images of the same person, a similar‑looking person, or an unrelated face to exercise face‑matching logic.
- **Numeric fields**: ID numbers and expiry dates entered by the user; tests may use valid values or intentionally wrong ones to trigger mismatch paths.

Most other fields (email, username, phone) are generated dynamically by the Robot scripts.
